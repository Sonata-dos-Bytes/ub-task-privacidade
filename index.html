Política de Privacidade

O aplicativo UB Task foi desenvolvido por estudantes como um projeto acadêmico e é gratuito.

Este aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários. Nenhuma informação é enviada a servidores externos. Todos os dados são armazenados localmente no dispositivo do usuário.

O aplicativo também não utiliza serviços de terceiros que possam coletar informações.

Como o UB Task é apenas uma ferramenta para ajudar estudantes a organizarem suas atividades da faculdade, nenhuma funcionalidade sensível ou relacionada à saúde está presente.
